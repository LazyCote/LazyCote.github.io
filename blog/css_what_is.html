Основы CSS. Что такое CSS?
CSS (Cascade Style Sheet) – это язык каскадных стилей, с помощью которого, вы можете как-либо графически, улучшить ваши страницу. То есть, этот язык позволяет сделать графический дизайн вашей страницы. “Начертить сетку” в виде flex боксов или grid сетку, и расставлять ваши блоки как вам угодно. Слово сетка, имеется ввиду не в прямом смысле, это будет невидимая сетка, которую вы со временем поймете и будите активно использовать. Есть ли смысл учить CSS ? ДА, учить его смысл есть, ибо на просто текстовые сайты заходить никто не будет. Вот вы бы, зашли на сайт где все идет в одну колонку и просто куча текста, даже если есть картинки, то всё-равно будет в колонку. Думаю вам бы такое не понравилось, поэтому при создании вам стоит использовать CSS. Но для того, чтобы его правильно использовать, было бы не плохо пройти селекторы в HTML&, а так же БЭМ разметку. На сколько сложно учить CSS? CSS довольно простой язык и разберется в нем даже ребенок, потому что в CSS вы пишете селектор(идетификатор) и в фигурные скобки записываете какой стиль вам нужен. Теперь давайте применим знания на практике, если вы не поняли на теории, то тут вы поймете точно. Давайте для начала создадим наш HTML скелет:
<!DOCTYPE html>
<html>
<head>
<meta charset=”utf-8”>
<title>Основы CSS</title >
</head>
<body>
<div class='div' >Тут class </div>
<div id='div' >Тут id </div>
</body>
</html>
 Давайте теперь создадим нашу кожу на созданный скелет. Создаем файл CSS с название index.css. После, мы должны привязать наш CSS к нашему HTML, это сделает строка кода <link rel="stylesheet" href="index.css">, объяснение того, какой атрибут для чего нужен был в первой статье про HTML.
В CSS есть 2 вида селекторов для class и id:

Class - это атрибут, он используется для группы или для одиночных элементов, блоков в вашем HTML файле. Пример <div class="any_name"> </div>, где any_name любое имя которое вы придумаете.
Id – это атрибут, использовать атрибут можно где угодно, но у него есть одно главное отличие от class. В отличие от того же class, имя определенного id может быть использовано только 1 раз, в то время, как имя класса может быть присвоено большой группе элементов. Пример <div id="unique_id_name" > </div>.
Так же, имена классов и ид могут совпадать, <div class=" unique_id_name " id="unique_id_name" > </div>. Такое часто можно встретить, когда стили ты подключаешь в CSS по классу, а манипуляцию над элементов в JS делаешь по id. Записывать одновременно класс и ид не запрещается.
Мы знаем как записать класс и ид, знаем их различая, но всё это в HTML. Как использовать эти знания в CSS ? Всё просто, давайте рассмотрим данный пример CSS кода:

.div {} /*Подключение по class*/
#div{}/*Подключение по id*/

Две данный строчки различаются тем, что они подключается к нашему элементу разными способами. Первая строчка подключается с помощью селектора по class, а вторая по id. В скобках, которые мы открываем, мы записываем наши манипуляции над блоками, которые были представлены в каркасе выше. Давайте изменим цвет у первого блока на синий, а у второго на красный. Делается это с помощью функции background-color или с помощью универсальной функции background. Если хотите узнать разницу, то переходите по второй ссылке на справочник htmlbook.ru. Учтем еще небольшие моменты, в функциях, где мы заменяем цвет, можно использовать разные методы ввода цвета, то есть, мы можем использовать палитру цветов HEX, RGB, RGBA, зарезервированное слово в CSS.
В данной примере мы рассмотрим варианты использования HEX и зарезервированного слова, чтобы вы могли подобрать цвет, то используйте сайт csscolor.ru.

.div {
background-color:#0008ff; /*HEX*/
}/*Подключение по class*/
#div{
background:red;/*зарезерв
ированное слово*/
}/*Подключение по id*/
