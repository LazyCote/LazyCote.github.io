Создание навигации
Сегодня мы поговорим о такой важной теме, как создание навигации на вашем сайте. Этот экскурс будет коротким, так что приступим. Для создания навигации нам понадобится тег header и тег nav,так же если у вас есть желание, то можно создать нижнюю панель навигации footer. Перейдем к объяснению каждого тега.
header – это тег шапки, на странице он может быть использован не один раз. Грубо говоря про его функционал, это тот же самый div, только с пометкой на читабельность кода.
footer – это тег подвал, он отвечает за нижнюю часть нашей страницы. Повторюсь, тоже самое что div, только с привязкой на удобство чтения и привязкой к нижней части нашей страницы.
nav – это специальный тег, которым мы обозначаем навигацию на нашем сайте. По принципу работы он div.
Перейдем к практике, создаем привычный нам файл html, надеюсь вы конструкцию уже запомнили, если нет загляните в прошлые уроки. Для начала нам нужно указать, что наша навигация будет в голове сайта, поэтому создаем header и в него сразу же засовываем nav.
<header>
<nav>
</nav>
</header>

 Зададим им классы, для последующей стилизации в CSS, а так же создадим список, на основе которого и будет наша навигация.
<header class='header'>
<nav class='header__nav'>
<ul class='nav_ul'>
<li class='nav_ul-items'><a href='#'>Главная </a></li>
<li class='nav_ul-items'><a href='#'>Товары</a></li>
<li class='nav_ul-items'><a href='#'>О нас</a></li>
</ul>
</nav>
</header>

 Если вы не знаете что такое ul и li, то в самом конце есть ссылки на справочники.
Вкладываем в элемент списка ссылку, чтобы мы отсылались на элементы нашего сайта. Вы так же можете сделать класс и у ссылки(тег <а/>) чтобы стилизовать ее. Как вы видите, наше меню расположено вертикально, а что если я хочу его сделать горизонтально ?
Сейчас мы познакомимся с flex сеткой, в этой статье мы пробежимся по ней, подробной о том как работает flex-box вы можете знать из нашей статьи про CSS flex-box. Для начала давайте отключим метки у нашего списка и сделаем все наши главные элементы горизонтально, для этого создаем файл index.css подключаем его и пишем следующий код:

.header {
max-width: 1920px;
width:100%;
margin:0 auto;
}
.header_nav {
width: 100%;
display:flex;
margin:0 auto;
background: red;
}
.nav_ul {
display:flex;
justify-content:space-around;
align-items:center;
}
.nav_ul-items {
list-style-type: none;
}

 Вот мы и сделали простое горизонтальное меню, вам останется добавить только ссылки на ваши остальные вкладки и всё. Принцип создания навигации в нижней части вашего сайта такой же, только использовать тег footer. developer.mozilla.org htmlbook.ru
