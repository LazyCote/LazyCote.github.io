Типы данных, массивы, функции
Для начала давайте познакомимся с тем, как стоит объявлять переменные в JavaScript, а затем с основными типами данных, их всего не много поэтому запомнить их будет довольно просто. Для объявления переменной существует ключевое слово var и let. В чем же их разница ? Если говорить проще, то переменная let, она грубо говоря локальна, а переменная var она более полномочная, но еще есть способ объявления переменных через window, это укажет что эта переменная имеет глобальное значение. Но этого делать не стоит, так как это в нашем деле бесполезно, по большей части, да и в добавок чтобы не перегружать глобал элементы.
Пример объявления переменной:
var name;
В данном примере вы указали переменную пустой. То есть, мы можем указать значение переменной сразу или же сделать это потом, пример:
var name;
name=5;

Переменная может иметь любой тип данных и изменяться на любой другой, как со строки в целое число, так из строки в число с плавающей точкой, так и в булеву переменную, но об этом чуть позже. Есть еще такие замечательные вещи как константы, создаются они ключевым словом const. Как надеюсь вы знаете, что константа, это такое значение которое никогда не изменяется, то есть, если переменную вы со временем изменить сможете, то константу нет, при ее попытке у вас ничего не выйдет, а также консоль может указать на то что вы пытаетесь сделать невозможное, пример:
var str=”Hellow”;
const PI=3.14;
str=25;

С константой такого не получится.
Давайте расскажу теперь про основные типы данных, а потом напишем нашу первую программу, ну или не первую если вы копируете код выше.
bool(буль, Булево значение) – это логическое значение которое использует true(истина) или false(ложь). В основном используется для построения логики.
string(строка) – это строковое значение, строки в JS записываются в ковычках, есть 2 варианта, ‘string1’ и “string2”, так же есть такой вариант `string3`, разные ковычки, но это всё строки.
number(integer, целочисленное значение) – это число, которое имеет целое значение, такие как 5, 10, 15, 25, 100, 245, 148 и другое любое число у которого нет точки.
float(число с плавающей точкой) – это число типа 3.14, 9.8 и тому подобные.
null – имеет значение null, коротко и понятно.
undefined – присваивается переменной при создании, если вы на задали ей значение или же если такой элемент не был обнаружен.
Как по мне, удобство JS заключается в том, что этот язык является динамическим по сравнению с тем же C#. Что я под этим подразумеваю ? Самый простой пример это те же переменные, в C# и C подобных языках нужно указывать конкретный тип данных и изменить ты его не сможешь, а вот JS, является динамическим, то есть ты просто создаешь переменную и можешь ее по 100 раз менять в разные типы данных.
Давайте теперь посмотрим все примеры типов данных, говорю сразу, имя у переменной может быть абсолютно любым, можете указывать его другим чтобы убедиться, но, вы не можете указывать зарезервированные слова по типу null, undefined, false, true и множество других.

Пример:
var str = “Это строка”;//Строка
var number = 10;//Целое число
var float = 2.25;//Число с плавающей точкой
var null = null;//Нуль
var undefined = undefined;//Ундефинед

const PI = 3.14;//Константа, по негласному правилу(хорошему тону), константы стоит писать заглавными буквами
 //Комментарий для строки
/* Комментарий для большого кол-ва строк */
Вот мы рассмотрели все основные типы данных, а так же комментарии, если не хотите путаться в коде, то делайте с помощью них пометки, компилятор их не видит.
Но всё-таки, вот я указал, а как мне посмотреть их значения допустим в консоли или уведомлении ? Для этого есть 2 метода:
1. console.log(переменная, строка и что-либо другое что хотите вывести)- выводит нужное вам значение в консоль, то что нужно вывести нужно писать в скобках.
2. alert() – выводит уведомление, которое на время морозит страницу.
С первой частью урока мы закончили, перейдем к более сложной части.
Массивы и функции.
Что есть массив ? Массив это какой-либо набор данных, которые имеют свое место, свой номер, свое имя и свое значение. Давайте рассмотрим синтаксис массива, пример:
var arr=[];
 Array – от англ. массив. Давайте разберем эту запись по кускам:
var – ключевое слово для создания переменной.
arr – название нашего массива.
[] – тело массива, в него записываются значения, через запятую. В JavaScript массив может принимать разнообразные типы.
Для того чтобы обратиться к элементу массива используется такая штука как индекс. Индекс – это цифровое обозначение элемента массива, отсчет начинается с нуля, то есть, первый элемент будет нулевым, а второй первым, не так уж и сложно.
Для заполнения массива у нас есть несколько методов:
1. Создать его пустым и заполнить после.
2. Заполнить сразу
Давайте воспользуемся обоими методами:
var arr=[15,3.14,”Hello”];
arr[3]=”World!”;

Теперь давайте выведем это в консоль:
console.log(arr);//чтобы вывести весь массив
console.log(arr[2]+arr[3]); //чтобы вывести несколько элементов

 Чтобы вывести несколько элементов в консоль используется знак + или ,.
Что ж, с массивами разобрались, но, у массива есть еще свои методы, давайте расскажу про основные, а вы сами с ними поиграетесь.
Но для начала расскажу как применяются методы, методы для любого элемента/переменной, записываются через точку, то есть arr.shift().
Методы:
arr.shift() – удаляет первый элемент массива.
arr.push(#) – добавляет #-значение в массив
arr.pop() – даляет последний элемент массива.
arr.unshift(#) – добавлет #-значение в начало массива
arr.reverse() – переворачивает массив
arr.sort() – сортирует массив
arr.slice(1,2) – разрезает массив создавая новый, 1 от какого индекса, 2 по какой
arr.splice(i) – удаляет элемент массива по индексу
arr.length – показывает длину массива
delete arr[i] – удаляет элемент по индексу, но минус этого способа заключается в том, что остаются дыры в массиве.
arr.concat() – складывает все элементы массива

Перейдем к последней теме, функции.
Что такое функция ? Простым языком, функция это блок с кодом, который начинает работать, когда вы его вызываете, в функции может быть всё что угодно, но у функции есть своя область видимости, то есть, переменные которые были созданы в функции вне ее показываться не будут. Удобство функции заключается в том, что можно сделать подобие модульной системы, где каждая функция будет отвечать за что-либо свое. Так как же создается функция?
Есть несколько видов функций:
1. Анонимная
2. С именем(как переменная)
3. Стрелочная(тоже самое что и анонимная, только запись кароче)
4. Самовызывающаяся, думаю из названия понятно, она анонимная но она вызывает саму себя, но думаю как минимум с написанием этого слова я напортачил).
Функция создается ключевым словом function;
Рассмотрим записи каждой из такой функции:
function func () {}

 Обычная функция которая имеет имя, принимает аргументы в круглые скобки и имеет тело функции, фигурные скобки, в которое и записывается код. Для вызова функции стоит указать ее имя и скобки круглые, то есть func();
Так же у этой функции есть второй тип записи:
var fung=()=>{};

Стрелочная функция:
()=>{}

Стрелочная функция в основном используется в методах, которые принимают функции для быстрого записи кода.
Анонимная:
Function () {}

 Это самая обычная функция только она не имеет имени.
(function(){})();

 Самовызывающаяся, вызывается сама, все хорошо.
Давайте напишем простенькую программу на основе обычной функции:
Function func (name) {
console.log(name);
}
func(‘Виктор’);

Данная функция принимает аргумент name, который мы можем указать при ее вызове. Аргументы, проще говоря, это те же самые переменные, только их значение зависит от того что вы укажете при их вызове.
