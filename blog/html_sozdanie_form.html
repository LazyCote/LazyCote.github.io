Создание форм.
Для начала давайте определим, для чего мы можем использовать на сайте формы. Формы мы можем использовать для создания формы обращения к администрации, для создания поиска по сайту, для отправки сообщений, для регистрации/авторизации и чего только вашей душе угодно. Давайте рассмотрим основные теги и их основные атрибуты. Более подробно читайте во втором с справочнике снизу.
form – этот тег дает понять браузеру, что всё что внутри, относится к форме и поэтому, оно будет отсылать POST/GET запросы, более подробно о них в курсе по PHP.
input – задаем как наша форма будет выглядеть и какой у нее будет функционал.
lable – просто пометка с надписью, его мы использовать не будем .
Давайте теперь рассмотри основные атрибуты и кто за что отвечает:

type – указывается в input и задает ему определенный функционал, примеры: type=”#”, где # может быть: radio, submit, button, file, password, text. Сразу оговорюсь, что для создания кнопки которая будет отправлять наш запрос на обработку, лучше использовать submit. Поиграйтесь и посмотрите все примеры написанные выше, вариантов типа гораздо больше, о них читайте в справочнике.
action– атрибут который указывается в теге формы, он определяет какой файл будет обрабатывать запрос, если этот код находится в самом начале файла, то просто пишется #, если же это отдельный файл, то указывается как ссылка, пример:
action=’php/register.php’. name – задает имя нашим input, для того, чтобы можно было различать запросы. Для каждого input стоит создавать разные имена. name=’pass’ value – это вбитое значение, которое можно просто стереть. value=’Ваш пароль’ placeholder – то призрачное значение, его нельзя стереть, но оно пропадает как только вы начинаете вводить ваш текст. В основном используется для того, чтобы указать пользователю, что сюда надо вводить. Пример: placeholder=’Введите ваш пароль’ disable – просто отключает элемент формы.
required – это атрибут, который делает поле обязательным.
Перейдем от теории к практике. Создаем стандартный шаблон html и начинаем создавать простую форму с полем ввода логина, пароля и кнопку:
<form action=’#’ class=’form’ >
<input name=’login’ type=’text’ class=’text’ placeholder=’Введите ваш логин’>
<input name=’pass’ type=password class=’text’ placeholder=’Введите ваш пароль’>
<input name=’btn’ type=’submit’ class=’btn’ >
</form>

 Данный шаблон формы работать не будет, в данной стадии это просто болванка, так как для обработки данных нужно использовать PHP или AJAX.
