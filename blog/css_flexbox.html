Flexbox
Ну что ж комрады. Надеюсь прочтя первую статью про CSS, вы не просто пошли дальше читать статьи, а начали практиковаться, изучая разные методы и тд и тп. Ибо если вы этого не делали, то читая эту статью, вам будет трудно понять что такое Flexbox. Так что преждем чем читать, идите тренируйтесь.
Что же такое Flexbox и для чего она нужна ? Если говорить просто, то Flexbox была создана для того, чтобы мы могли удобно, просто и быстро располагать наши блоки. Flex устанавливается для контейнера(родительского элемента), который контролирует объекты внутри него. Если вы читали как сделать простую навигацию по сайту, то вы уже сталкивались с flex сеткой, так как ее там мы и использовали. Давайте разберем основные методы, которыми вы будете пользоваться по больше части. Тут только описание основных методов, что в них надо прописывать дополнительно, читайте в справочнике снизу.
display – задает блоку то, как он будет отображаться на экране, в нашем случае мы будем использовать значение flex, что даст понять браузеру что мы использует объект типа flex.
flex-direction – задает объекту flex значение отображения по оси x или по осу y. По стандарту стоит значение row, что означает, что дочерние элементы будут выставлены в строку. Если прописать column, то объекты будут расположены относительно оси y и будут располагаться в колонку.
justify-content – устанавливает положение относительно экрана, все зависит от оси. Если используется стандартное значение row, то корректировка будет по оси x, при значение column, ось x меняется на ось y.
align-items – тоже самое что и justify-content, только на оборот, где один в x, другой в y и на оборот.
align-self – тоже самое что и align-items, только корректировка ведется не для всех дочерних элементов, а для конкретного, к которому приделан этот метод.

Узнали об основных методах, что ж. Пора приступать к практике, создаем привычный нам файл html, файл css и подключаем его. Для начала давайте сделаем контейнер, в котором будет 1 блок, и попробуем его отцентровать. Затем проделаем тоже самое, только добавим два блока и расположим их относительно центра, поймете в чем разница. Затем снова 2 блока и сделаем их относительно краев файла. И самым последним этапом, мы сделаем пирамидку. По итогу у нас должно получиться 4 контейнера с разными результатами. Создаем первый контейнер:

<section class='first'>
<div class='first-item'></div>
</section>

 Далее пропишем стили для созданных блоков, но перед этим, сделаем сброс настроек файла:
/*
Убираем внутренние отступы
Убираем наружние отступы
Убираем то, что я забыл
Убираем наружнюю линий
*/
* {
padding: 0;
margin: 0;
box-sizing: 0;
outline: 0;
}
.first {
max-width: 1920px;
width: 100%;
height: 500px;
background: red;
margin:0 auto; /* Центруем родительский блок относительно центра экрана */
display: flex; /* Задаем род.блоку значение flex*/
justify-content: center; /* Центрует дочерний жлемент по центру относительно родителя*/
}
.first-item {
width: 100px;
height: 100px;
background: yellow;
}
 Самый простой пример работы flex сетки. Переходим ко второму этапу знакомства, расположим два блока относительно центра:

HTML:
<div class='second'>
<div class='second-item'></div>
<div class='second-item'></div>
</div>

 CSS:
.second {
max-width: 1920px;
width: 100%;
height: 500px;
margin:0 auto; /* Центруем родительский блок относительно центра экрана */
display: flex; /* Задаем род.блоку значение flex*/
justify-content: space-around; /*Делает опорной точной для расположения объектов центр*/
}
.second-item {
width: 100px;
height: 100px;
background: yellow;
}
 В третьем этапе будет практически тот же код, только вместо spase-around, который выравнивает относительно центра, у нас будет space-between , которые выравнивает относительно краев. Проделайте это сами и посмотрите разницу. Переходим сразу к нашей пирамидке:

HTML:
<div class='third'>
<div class='third-container'>
<div class='third-item'></div>
</div>
<div class='third-container'>
<div class='third-item'></div>
<div class='third-item'></div>
</div>
<div class='third-container'>
<div class='third-item'></div>
<div class='third-item'></div>
<div class='third-item'></div>
</div>
</div>

 CSS:
.third {
max-width: 1920px;
width: 100%;
height: 500px;
}
.third-container {
width: 100%;
margin: 0 auto;
display: flex;
justify-content: space-around;
}
.third-item {
width: 100px;
height: 100px;
background: yellow;
}

 Как вы видите у нас получилась самая нелепая пирамидка, потому что она маленько кривая, но, надеюсь вы заметили, что при добавлении элементов в контейнеры, блоки выставляются по разному. Этим она и удобна, что делает вещи автоматически и не придется выставлять параметры в ручную, пользуясь теми же margin/padding.
Ну вот мы вкратце познакомились с такой штукой как Flexbox, она довольно проста в освоении, в отличии от grid, но grid дает больше возможностей для стилизации вашей страницы
